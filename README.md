## AI-Based Phishing Email Detection

This project implements Logistic Regression and Random Forest models
to detect phishing emails using engineered email features.

### Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
- matplotlib
- seaborn

### Models
- Logistic Regression
- Random Forest

### Evaluation Metrics
Accuracy, Precision, Recall, F1-score, ROC-AUC
